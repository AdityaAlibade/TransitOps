# TransitOps Backend Boilerplate

TransitOps backend skeleton using Node.js, Express.js, TypeScript, PostgreSQL, and Prisma ORM.

## Setup Instructions

1. **Install Dependencies**
   ```bash
   cd server
   npm install
   ```

2. **Configure Environment Variables**
   - Copy `.env.example` to `.env`
   - Configure your local PostgreSQL database URL connection.

3. **Development Command**
   ```bash
   npm run dev
   ```

4. **Production Build**
   ```bash
   npm run build
   ```
